# KeyHero 

KeyHero is a browser-based guitar hero like game played with your keyboard built using TypeScript and the HTML5 Canvas API.

> **Note:** This project is currently in early development, and the project is not yet very polished.

##  Getting Started

Follow these steps to run the game locally on your machine.

### Prerequisites
* [Node.js](https://nodejs.org/) 
* npm

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Eetu-s/KeyHero.git](https://github.com/Eetu-s/KeyHero.git)
    cd KeyHero
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Play**
    Open your browser and navigate to the link shown in the terminal (usually `http://localhost:5173`).
